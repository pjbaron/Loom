# /loom-about - Get All Knowledge About Entity

Retrieve all notes, intents, and hypotheses about a specific entity.

## Usage
```
/loom-about <entity_name>
```

## Examples
- `/loom-about validate_user`
- `/loom-about CodeStore`
- `/loom-about process_request`

## Instructions

Run the Loom about command:

```bash
/mnt/f/experiments/Loom/loom about "$ARGUMENTS"
```

The output includes:
- All notes mentioning the entity
- Intent documentation
- Related hypotheses (confirmed and unconfirmed)
- Historical debug findings
