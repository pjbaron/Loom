# /loom-class - Explain a Class

Get a detailed overview of a class including all its methods, attributes, inheritance, and usage patterns.

## Usage
```
/loom-class <class_name>
```

## Examples
- `/loom-class CodeStore`
- `/loom-class Machine`
- `/loom-class World`

## Instructions

Run the Loom class explanation:

```bash
/mnt/f/experiments/Loom/loom class "$ARGUMENTS"
```

The output includes:
- Class docstring and purpose
- All methods with signatures
- Inheritance hierarchy
- Key attributes
- Usage examples from the codebase
