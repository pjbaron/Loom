# /loom-todo-next - Get Next TODO

Get the next TODO item to work on (highest priority, FIFO order).

## Usage
```
/loom-todo-next
```

## Instructions

Run the Loom todo next command:

```bash
/mnt/f/experiments/Loom/loom todo next
```

Shows:
- TODO ID
- Title and full prompt
- Tags and context
- Related entities/files

After starting work, use `/loom-todo-start <id>` to mark it in progress.
