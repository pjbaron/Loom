# /loom-todo-stats - TODO Queue Statistics

Get quick stats on the TODO queue.

## Usage
```
/loom-todo-stats
```

## Instructions

Run the Loom todo stats command:

```bash
/mnt/f/experiments/Loom/loom todo stats
```

Shows:
- Total pending items
- Items in progress
- Completed items
- Items by tag
- Critical items count
