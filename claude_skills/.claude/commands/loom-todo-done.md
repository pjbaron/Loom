# /loom-todo-done - Complete a TODO

Mark a TODO as completed with optional notes on how it was resolved.

## Usage
```
/loom-todo-done <id> [--notes "resolution notes"]
```

## Examples
- `/loom-todo-done 5`
- `/loom-todo-done 5 --notes "Fixed by adding null check in line 42"`

## Instructions

Run the Loom todo done command:

```bash
/mnt/f/experiments/Loom/loom todo done $ARGUMENTS
```

Resolution notes become part of the knowledge base for future reference.
