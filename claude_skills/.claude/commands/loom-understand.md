# /loom-understand - Semantic Code Search

Search the codebase for code matching a natural language query. Use this when you need to find code related to a concept, feature, or behavior.

## Usage
```
/loom-understand <query>
```

## Examples
- `/loom-understand authentication logic`
- `/loom-understand error handling`
- `/loom-understand database connection pooling`

## Instructions

Run the Loom understand command to find relevant code:

```bash
/mnt/f/experiments/Loom/loom understand "$ARGUMENTS"
```

The output will show:
- Matching functions, classes, and methods
- File locations and line numbers
- Relevance scores
- Code previews

Use this tool BEFORE making changes to understand existing patterns in the codebase.
