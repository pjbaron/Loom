# /loom-architecture - Codebase Architecture Overview

Get a high-level overview of the codebase architecture including module relationships, key abstractions, and system structure.

## Usage
```
/loom-architecture
```

## Instructions

Run the Loom architecture command:

```bash
/mnt/f/experiments/Loom/loom architecture
```

The output includes:
- Module dependency graph
- Core abstractions and their roles
- Layer structure (if applicable)
- Entry points and main flows
- Key design patterns used

Use this to understand the overall system before making architectural changes or adding new features.
