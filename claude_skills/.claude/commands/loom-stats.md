# /loom-stats - Codebase Statistics

Show statistics about the indexed codebase.

## Usage
```
/loom-stats
```

## Instructions

Run the Loom stats command:

```bash
/mnt/f/experiments/Loom/loom stats
```

The output includes:
- Total files, classes, functions indexed
- Lines of code
- Language breakdown
- Database size and freshness
- Coverage metrics
