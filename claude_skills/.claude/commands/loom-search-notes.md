# /loom-search-notes - Search Knowledge Base

Search all notes, intents, and hypotheses in the Loom knowledge base.

## Usage
```
/loom-search-notes <query>
```

## Examples
- `/loom-search-notes "race condition"`
- `/loom-search-notes "authentication"`
- `/loom-search-notes "null pointer"`

## Instructions

Run the Loom search command:

```bash
/mnt/f/experiments/Loom/loom search-notes "$ARGUMENTS"
```

The output shows matching notes with:
- Note ID (for use with /loom-resolve)
- Content preview
- Related entities
- Creation date
