# /loom-module - Explain a Module

Get a structured overview of a module's contents including all classes, functions, and their relationships.

## Usage
```
/loom-module <module_name>
```

## Examples
- `/loom-module codestore`
- `/loom-module tracer`
- `/loom-module terrain`

## Instructions

Run the Loom module explanation:

```bash
/mnt/f/experiments/Loom/loom module "$ARGUMENTS"
```

The output includes:
- Module purpose and docstring
- All public classes and functions
- Internal structure and organization
- Dependencies and imports
